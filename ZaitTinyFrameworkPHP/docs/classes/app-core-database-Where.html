Classe Where

Essa classe é responsável por gerar cláusulas WHERE em SQL.

Exemplo de uso:

$where = new Where();
$where->addCondition('AND', 'nome', '=', 'João');
$where->addCondition('OR', 'idade', '>', 30);
$query = $where->build();
echo $query; // Produz: WHERE nome = ? AND idade > ?
